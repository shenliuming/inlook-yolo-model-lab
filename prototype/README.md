# 沈柳名的AI 实验室前端

前端使用 Vue 3 + Vite，不使用 React。当前版本已经接入本地 FastAPI 后端，支持真实模型列表加载、图片/视频上传、结果预览、结果下载和报告展示。

## 启动方式

```bash
npm install
npm run dev
```

默认开发地址：

- `http://127.0.0.1:5173`

## 开发代理

Vite 已配置以下代理：

- `/api` -> `http://127.0.0.1:8000`
- `/outputs` -> `http://127.0.0.1:8000`
- `/reports` -> `http://127.0.0.1:8000`

## 页面能力

- 页面加载时自动请求 `/api/health` 和 `/api/models`
- 支持图片识别 / 视频识别切换
- 支持模型切换
- 支持原始素材预览
- 支持识别结果图片 / 视频预览
- 支持下载结果文件和查看报告 JSON
- 支持 CPU 提示和错误日志展示
